Battleships Project

As part of my Computer Science A-Level, I had to work on an extended programming project. This involved me having to conduct research about already existing products and then formulate my own designs. 

I decided to create a version of Battleships, which could be played on desktop PCs. By using a variety of different data structures, file handling and an Insertion Sort algorithm - I was able to create a version of Battleships. Players are able to play against the computer (CPU) or against another player (via local co-op). Furthermore, players can compete to try earn the highest score to earn their place on the leaderboard, and players can save their games and load them to continue playing another time.
